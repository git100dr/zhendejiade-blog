# 欢迎使用Mditor

Mditor是一个*轻量级*的**markdown**编辑器。取名自markdown + editor，用于实现页面markdown输入框的便利操作。

## Markdown是什么

> Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档，然后转换成格式丰富的HTML页面。 —— [维基百科](https://zh.wikipedia.org/wiki/Markdown)

```jsx
hello,  jsx code.
```

```js
world !--js code
```

**快捷键：**

- **加粗文字** Ctrl + B
- _斜体文字_ Ctrl + I
- [链接文字]() Ctrl + L

**列表：**

3. ......
